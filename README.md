# 🛒 A-6 Digital Tools Buying Website

A modern and responsive web application where users can explore and purchase digital tools. The platform allows users to browse products, add them to a cart, and manage their selections with real-time updates and notifications.

## ✨ Features

* 🛍️ **Product Listing System** – Browse 6–10 digital tools displayed in a clean 3-column layout
* 🔄 **Toggle Functionality** – Switch between Products and Cart views easily
* 🛒 **Cart Management** – Add, remove, and clear products with real-time updates
* 🔢 **Dynamic Cart Count** – Navbar updates instantly when products are added/removed
* 🔔 **Toast Notifications** – Alerts for add to cart, remove, and checkout actions
* 💰 **Checkout System** – Clears all cart items on proceed to checkout
* 📱 **Fully Responsive Design** – Optimized for mobile, tablet, and desktop

## 🛠️ Technologies Used

* **React.js** – Frontend framework
* **Tailwind CSS** – Styling
* **DaisyUI** – UI components
* **JavaScript (ES6+)** – Logic & functionality
* **React-Toastify** – Notifications
* **JSON** – Product data

---

## 📦 Product Data Structure

Each product includes:

* `id`
* `name`
* `description`
* `price`
* `period` (one-time / monthly / yearly)
* `tag`
* `tagType`
* `features` (array)
* `icon`

---

## 🛒 Cart Functionality

* Add products to cart
* Remove individual products
* View selected products in a single-column layout
* See total product count in navbar
* Proceed to checkout clears all items
