Digital Tools Buying Website

Description:
A responsive web app to browse and buy digital tools. Users can view products, add them to a cart, and checkout with real-time updates and notifications.


Features:
- Browse 6–10 products in a 3-column layout
- Add/remove products from cart with dynamic cart count
- Toast notifications for cart actions (React-Toastify)
- Proceed to checkout clears all items
- Fully responsive design

Technologies Used:
React.js, Tailwind CSS, DaisyUI, JavaScript (ES6+), React-Toastify, JSON

Cart Functionality:
- Add/remove products
- See total count in navbar
- Checkout clears cart
